# User Stories

1. As a user, I want to register so that I can create an account.
2. As a user, I want to login so that I can access my dashboard.
3. As a user, I want to view items on home screen.
4. As a user, I want to see item details.
5. As a user, I want to save favorites.
6. As a user, I want settings options.
7. As a user, I want notifications.
8. As a user, I want API data displayed.
9. As a user, I want my data saved locally.
